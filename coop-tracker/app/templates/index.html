      Set up a nesting box

      Enter the box's inside width — the app measures eggs against this instead of needing a coin placed in every photo. You'll then take a few photos so it can learn to spot this box's edges reliably.

      Training photos

      Every photo the model learns from. Tap Edit to re-correct one, or Remove to exclude it from training. Changes take effect the next time you train.

      Backup & Restore
